# cwrs_go_server

#### Description
cwrs_go_server 是一个基于go语言开发的web服务框架。
基于gin框架，使用jwt作为token验证框架、gorm作为数据库操作框架、使用viper作为配置文件框架、zap作为日志框架、swagger作为api文档。
